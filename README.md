# naschi.at ![[Build Status]](https://github.com/naschidaniel/naschi/workflows/release/badge.svg)
My Personal Website. Visit [naschi.at](https://naschi.at) for a live preview of the project.

## Dependencies

Make sure those are available on your system:

* nodejs
* npm

## Build project

Install dependent node modules.

```bash
$ npm install
```

## Development environment

Serve with hot reload at localhost:3000
```bash
$ npm run dev
```

## Lint
```bash
$ npm run lint
```
or fix lint problems.
```bash
$ npm run lintfix
```

## Build for production and launch server

```bash
$ npm run build
$ npm run start
```

## Generate static project

```bash
$ npm run generate
```
