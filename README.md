## Project setup
```
yarn
```

### Development
#### Compiles and hot-reloads for development
```
yarn dev
```

#### Lints and fixes files
```
yarn lint
```

### Production
#### Compiles and minifies for production
```
yarn build

# preview of latest build
yarn serve
```
